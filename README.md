# Animation Editor

## Development Environment

- Node v20.10.0
- Nuxt v3.14
- Konva v9.3.16

## Features

- 鍵盤：刪除、方向鍵移動。
- 游標：拖曳、群組選取、cmd + 滾輪放大縮小。
- 動畫時間軸：使用 Gsap.timeline。
